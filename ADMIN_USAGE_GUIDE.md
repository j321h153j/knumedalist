# 행운제 운영 관리 페이지 사용법

이 문서는 운영진이 `admin.html` 페이지로 경기/부스 결과를 입력하는 방법을 정리한 안내서입니다.

## 접속과 로그인

1. 운영 관리 페이지에 접속합니다.
2. 발급받은 운영진 아이디와 비밀번호를 입력합니다.
   - 아이디는 `futsal1`처럼 앞부분만 입력해도 됩니다.
   - 전체 이메일 형식인 `futsal1@test.local`로 입력해도 됩니다.
3. 로그인에 실패하면 아이디 또는 비밀번호를 다시 확인합니다.
4. 다른 계정으로 테스트해야 할 때는 오른쪽 위 `로그아웃`을 누른 뒤 다시 로그인합니다.

## 운영진 아이디

로그인할 때는 아래 표의 `로그인 아이디`만 입력해도 됩니다. 예를 들어 `basketball1@test.local` 계정은 `basketball1`만 입력해도 로그인됩니다.

비밀번호는 별도로 전달받은 운영진 공통/개별 비밀번호를 사용합니다.

## 권한별 화면

### 일반 운영진

일반 운영진은 아래 메뉴를 봅니다.

- `대시보드`: 전체 현황과 점수 확인
- `내 할일`: 본인에게 배정된 경기/부스 입력
- `점수판`: 학번별 종합 점수 확인

일반 운영진은 본인에게 배정된 경기 또는 부스만 입력할 수 있습니다. 다른 경기 결과는 읽을 수 있지만 수정할 수 없습니다.

### 대표 운영진

대표 운영진은 아래 메뉴를 봅니다.

- `대시보드`: 전체 현황 확인
- `결과 관리`: 모든 경기/부스 결과 입력 및 수정
- `점수판`: 학번별 종합 점수 확인

대표 운영진은 전체 경기와 부스를 수정할 수 있습니다. 자동 대진이 애매한 경우 수동 대진 결정도 대표 운영진만 할 수 있습니다.

## 경기 결과 입력

1. 일반 운영진은 `내 할일`에서 담당 경기를 선택합니다.
2. 대표 운영진은 `결과 관리`에서 종목을 펼친 뒤 원하는 경기를 선택합니다.
3. 경기 시작 전이면 `시작` 버튼을 누릅니다.
4. 확인 팝업이 뜨면 내용을 확인하고 진행합니다.
5. 결과 입력 폼이 나오면 점수를 입력합니다.
6. `결과 저장` 버튼을 누릅니다.
7. 저장 확인 팝업에서 확인하면 결과가 서버에 저장됩니다.
8. 저장이 성공하면 목록 화면으로 돌아갑니다.

## 종목별 입력 방식

### 농구 / 풋살

- 양쪽 팀의 정규 점수를 입력합니다.
- 동점일 때만 자유투 또는 승부차기 입력 칸이 나타납니다.
- 농구는 자유투 점수, 풋살은 승부차기 점수를 입력합니다.
- 승자는 서버가 자동 계산합니다.

### 줄다리기

- 세트별 승자와 시간을 입력합니다.
- 3판 2선승 기준입니다.
- 예선 승자 비교는 서버가 자동 계산합니다.

### 피구

- 세트별 양 팀 생존자 수를 입력합니다.
- 세트별 생존자 수가 같으면 승자를 확정할 수 없으므로 저장되지 않습니다.
- 피구의 예선 승자 비교는 패 수와 생존자 합계 기준을 사용합니다.

### 계주

- 시간 기록은 입력하지 않습니다.
- 1등부터 5등까지 학번 순위만 선택합니다.
- 같은 학번을 중복 선택하면 저장되지 않습니다.

## 대진 자동 배정

예선 결과를 저장하면 서버가 다음 대진을 자동으로 계산합니다.

- 예선 승자 중 더 강한 팀은 결승으로 이동합니다.
- 예선 승자 중 약한 팀은 부결승으로 이동합니다.
- 부전승 팀은 부결승의 반대편에 배정됩니다.
- 부결승 승자는 결승으로 이동합니다.

농구/풋살은 `득실차 → 다득점 → 자유투/승부차기 득실차 → 자유투/승부차기 다득점` 순서로 비교합니다.

자동 판단이 불가능한 완전 동률이면 대표 운영진 화면에 수동 대진 결정 버튼이 표시됩니다. 수동 결정에서는 부결승으로 보낼 팀을 선택하면, 남은 팀은 결승으로 자동 배정됩니다.

## 경기 결과 수정

1. 입력 완료된 경기에서 `수정` 버튼을 누릅니다.
2. 확인 팝업에서 수정 여부를 확인합니다.
3. 기존 점수가 들어간 폼이 열립니다.
4. 필요한 값을 수정한 뒤 `결과 저장`을 누릅니다.
5. 저장하면 다음 대진과 점수판이 다시 계산됩니다.

주의: 예선 결과를 수정하면 부결승/결승 대진도 바뀔 수 있습니다. 이미 뒤 경기 결과를 입력한 상태라면 대표 운영진이 반드시 확인해야 합니다.

## 부스 기록 입력

부스 담당자는 `내 할일`에서 담당 부스를 선택합니다.

### 웨이트 챌린지 / 데드리프트

- 이름을 입력합니다.
- 학번 팀을 선택합니다.
- 기록을 kg 단위로 입력합니다.
- 한 명씩 저장합니다.
- 저장된 기록은 아래 목록에서 확인할 수 있습니다.
- 잘못 입력한 기록은 목록의 `수정` 버튼으로 수정합니다.

### 신발 던지기

- 이름을 입력합니다.
- 학번 팀을 선택합니다.
- 점수를 입력합니다.
- 최대 점수는 50점입니다.
- 50점을 넘으면 저장되지 않습니다.
- 한 명씩 저장합니다.
- 잘못 입력한 기록은 목록의 `수정` 버튼으로 수정합니다.

### 경찰과 도둑

- 타임 스케줄은 서버 데이터 기준으로 표시됩니다.
- 각 타임은 `예정 / 진행 중 / 종료` 상태로 보입니다.
- 여러 타임이 동시에 진행 중으로 보이거나 시간이 `-`로 보이면 데이터 설정 문제이므로 대표 운영진에게 알려야 합니다.

## 점수판 확인

`점수판` 메뉴에서 학번별 종합 점수를 확인할 수 있습니다.

- 학번 카드를 누르면 해당 학번이 어떤 종목/부스에서 몇 점을 얻었는지 펼쳐서 볼 수 있습니다.
- 경기 결과나 부스 기록을 수정하면 점수판은 서버에서 다시 계산됩니다.

## 저장이 안 될 때

아래 순서대로 확인합니다.

1. 인터넷 연결 상태를 확인합니다.
2. 새로고침 후 다시 시도합니다.
3. 로그인 세션이 풀렸다면 다시 로그인합니다.
4. 담당 권한이 없는 경기/부스인지 확인합니다.
5. 필수 입력값이 빠졌는지 확인합니다.
6. 동점인데 자유투/승부차기 점수를 입력하지 않았는지 확인합니다.
7. 신발 던지기 점수가 50점을 넘었는지 확인합니다.

계속 실패하면 화면에 뜬 오류 문구와 어떤 경기/부스를 입력 중이었는지 대표 운영진에게 전달합니다.

## 운영 중 주의사항

- 한 경기 결과는 담당자 한 명이 최종 입력하는 것을 권장합니다.
- 같은 경기 화면을 여러 사람이 동시에 수정하지 않도록 합니다.
- 저장 버튼을 누른 뒤 화면이 바뀔 때까지 기다립니다.
- 저장 후에는 목록 또는 점수판에서 결과가 반영되었는지 확인합니다.
- 결과를 잘못 입력했을 때는 새로 입력하지 말고 기존 결과를 수정합니다.
